# Express JS MVC

Express JS dengan struktur MVC.

## Instalasi
Jalankan perintah dibawah ini
```bash
  npm install
```

## Run Server
Jalankan perintah dibawah ini
```bash
  npm run serve
```
## Includes

- Configurasi .env
- Vue Good Table Server Side
- JSON Web Token
- Sequelize ORM
- Nodemailer
- Express Validator
- Crypto
- Multer
